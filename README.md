# kathy.pm

## Running Locally

```bash
$ git clone https://github.com/simpsoka/kathy-pm-next.git
$ cd kathy-pm-next
$ yarn
$ yarn dev
```

## Built Using

- [Next.js](https://nextjs.org/)
- [Vercel](https://vercel.com)
- [MDX](https://github.com/mdx-js/mdx)
- [Tailwind CSS](https://tailwindcss.com/)
